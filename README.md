# lampinstall
This is an automated script to install Apache, Mysql and PHP on Ubuntu 17

Instructions:


1- change to root
  
  sudo su
  
  
2- clone the git
  
  git clone https://github.com/arvage/lampinstall
  
  
3- go to downloaded directory and make the file executable
  
  cd lampinstall
  
  chmod +x lapminstall.sh
  
  
4- run the script
  
  ./lampinstall.sh
